Global Campus Connect is a modern, high-performance web application built with Vite, React, TypeScript, and TailwindCSS, designed to streamline communication and connectivity across campuses with a clean, modular architecture. 

To run the project locally, simply clone the repository, install dependencies using 
npm, pnpm, or bun, and launch the development server via npm run dev to access the app at http://localhost:5173/. The project includes a scalable folder structure, production-ready configurations, ESLint for consistent code quality, and commands for building (npm run build) and previewing (npm run preview) optimized builds, ensuring a smooth developer experience from setup to deployment.
